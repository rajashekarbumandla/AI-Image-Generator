# AI-Powered Text-to-Image Generator 

This project implements an **AI-powered text-to-image generator** using open-source Stable Diffusion models.  


Users can:
- Enter text prompts
- Choose visual style (photorealistic / artistic / cartoon)
- Adjust generation parameters
- Generate multiple images per prompt
- Download images in PNG/JPEG formats
- View metadata for each generation

All images include a watermark indicating AI-origin and simple safety filters are applied.
